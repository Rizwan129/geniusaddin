1) EXPENSE

Calcutates the expense under given category from startDate till endDate inclusive.
If category is empty then total expense for given period is fetched. If endDate is empty then it is assumed to be same as startDate.

Parameters:
  • category
  • startDate (YYYY-MM-DD) or (mmm-YYYY)
  • endDate (YYYY-MM-DD) or (mmm-YYYY)

=EXPENSE("Job Expenses", "2018-01-01", "2020-02-01")

=EXPENSE("Job Expenses", "jan-2020", "nov-2020")

=EXPENSE("Job Expenses", "jan-2020",)

3) COGS

Calcutates the Cost of Goods Sold under a given category from the startDate till the endDate.
If the category is empty then the total Cost of Goods Sold for given period is displayed. If endDate is empty then it is assumed to be same as startDate.The formula has to match the phrasing from the List Categories tab exactly in order to work.

Parameters:
  • category
  • startDate (YYYY-MM-DD) or (mmm-YYYY)
  • endDate (YYYY-MM-DD) or (mmm-YYYY)

=COGS("COGS", "2018-01-01", "2020-02-01")

=COGS("COGS", "jan-2020", "nov-2020")

=COGS("COGS", "jan-2020",)
